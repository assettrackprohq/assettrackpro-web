# AssetTrack Pro — Field App

**AssetTrack Pro** is a high-performance, mobile-optimised web interface designed specifically for field operations, asset auditing, and real-time inventory management. It provides a streamlined, professional workflow for field officers to capture, verify, and track physical assets in diverse environments.

---

## 🚀 Features

* **Mobile-First Interface:** Optimized for field use with a responsive viewport (393x852px) to ensure usability on handheld devices.
* **Dual-Theme System:** Includes "Light Sky-Blue" and "Dark Ocean" modes, allowing users to switch based on lighting conditions.
* **Integrated Barcode Scanner:** Features a built-in scanner overlay with a live video feed and framing guide for rapid asset identification.
* **Advanced Audit Workflow:** A dedicated "Audit Scan Zone" with progress tracking and real-time status verification.
* **Comprehensive Data Capture:** Structured input forms for asset descriptions, serial numbers, and photo attachments.
* **Operational Dashboard:** A central hub displaying session statistics like "My Captures," "Audited" counts, and system sync status.
* **Diagnostic Tools:** Includes an internal error log and "spooling" management to ensure data integrity during field syncs.
